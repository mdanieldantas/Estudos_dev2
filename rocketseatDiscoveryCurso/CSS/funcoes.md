# funções 

* nome seguido de abre e fecha parenteses 
* recebe argumentos 

## Exemplos 

```CSS

@import url("http://urlaqui.com/style.css")

{
color: rgb(255, 0, 100);
width: calc(100% - 10px);
}

'''

Em programação, funcões são reconhecidas por causar um reaproveitamento de código.

* rgb()
* hsl()
* url()
* calc()
